The :has() selector

The :has() selector checks whether a parent contains another element inside it.

The element to check for can be specified in the has() function.

This is an H2

This section is style based on the :has() function.

This style only affects section element that has an h2 element.

Note that the has function can detect a child of any level, no matter how deep.
