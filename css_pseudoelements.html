Pseudoelements

First Letter

Pseudoelements allow you to modify specific parts of existing elements.

One of these pseudoelements is called ::first-letter which modifies the first letter of an element.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Fugit laborum consectetur error? Deserunt neque consectetur, voluptatibus totam assumenda qui, suscipit optio eaque modi aspernatur magnam laboriosam possimus consequuntur omnis blanditiis.

First Line

Another pseudoelement is the ::first-line, and it modifies the first line of an element instead of just the first letter.

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nesciunt quos laboriosam nulla sapiente vero debitis neque voluptas aliquid fugit, architecto adipisci saepe temporibus quod magni, numquam commodi repellat, blanditiis est!
